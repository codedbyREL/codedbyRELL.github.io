Python Exploration

Project Overview:
The project analyzes a dataset with metadata for 45,000 movies, focusing on relationships between features like budget, gross revenue, and ratings.
Data Cleaning:
Rows with missing 'budget' and 'gross' values were dropped. 'Budget' and 'gross' were converted to int64. A new 'correctyear' column was created from the 'released' column.
Exploratory Data Analysis (EDA):
A Seaborn scatter plot explores the correlation between budget and gross revenue.
Key Features:
Columns include movie details, ratings, release information, cast, budget, gross revenue, and production details. The target variable is 'gross.'
Findings:
- Scatter plot investigates the correlation between budget and gross revenue.
- Distribution analysis provides insights into budget and gross revenue trends.
- Temporal trends can be explored using the corrected release year.
Next Steps:
- Statistical analysis for correlation coefficients.
- Deeper exploration of genres, directors, or production companies.
- Consideration of external factors like inflation.
Conclusion:
The project aims to uncover patterns in the film industry, laying the groundwork for a detailed analysis of movie success factors. Further analyses can provide a more nuanced understanding of the industry dynamics.
